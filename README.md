# Online Music Player

## Install Needed Modules
```shell
npm install
```

### Open A Terminal And Run Json Server For Watch
```shell
json-server --watch ./data/db.json
```

### Open A New Terminal And Run Vue Js Development Version
```shell
npm run serve
```

### Compiles And Minifies For Production
```shell
npm run build
```
